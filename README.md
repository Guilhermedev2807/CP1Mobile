## 🚀 Funcionalidades (Critérios de Avaliação)
O projeto cumpre 100% dos requisitos solicitados:

1.  **Exibir dicas aleatoriamente:** Ao iniciar, o app sorteia uma dica de uma lista pré-definida.
2.  **Atualizar dica ao clicar:** Botão funcional que gera uma nova dica aleatória ao ser pressionado.
3.  **Cores coerentes:** Design limpo utilizando paleta de cores moderna (Azul e tons de cinza claro).
4.  **Layout coerente:** Interface responsiva com uso de Cards, sombras e bordas arredondadas.
5.  **Código em Inglês:** Toda a estrutura de variáveis, funções e estilos foi escrita em inglês (Ex: `tipsList`, `generateRandomTip`).
